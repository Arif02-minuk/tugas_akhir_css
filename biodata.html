Tentang Saya

basic sekolah saya adalah pelayaran , namun saya sangat suka dengan dunia teknologi terutama dibidang komputer. diawal saya bekerja dipelayaran selalu menyempatkan untuk belajar coding melalui sebuah media social dan akhirnya melanjutkan di sebuah bootcamp bernama niomic saya juga menyelesaikan beberapa tugas dari beberapa bootcamp mengenai frontend. sekarang saya berfokus memperdalam ilmu frontend. yang kemudian masuk ke beckend dev saya memiliki target atau cita cita untuk menguasai fullstack dev.

Pendidikan

sekolah dasar lulus tahun 2009 sekolah menengah lulus tahun 2012 kemudian lanjut smk pelayaran lulus tahun 2015 awal ketertarikan dengan ilmu IT menjelang lulus kursus pelayaran waktu itu suka mengolah sebuah sistem rom pada hp kemudian semakin penasaran dengan bagaimana sistem rom dihp atau android dibuat. awalnya saya mempelajari bahasa c++ karena peralatan belum memadai saya mencoba bahasa lain akhirnya menemukan sebuah code dimana fungsi umumnya untuk membuat website , tertarik untuk memperdalam saya lanjut sampai saat ini sedikit menguasai html css dan javascript . untuk menunjang semua skill saya memutuskan untuk mengikuti sebuah bootcamp yang membuat saya tertarik yakni NIOMIC, sejauh ini saya belum membuat project nyata. semoga setelah menyelesaikan program di bootcamp tersebut sya bisa membuat project nyata.

Pengalaman

sejauh ini pengalaman di IT belum membuat suatu project nyata, akan tetapi saya pernah membuat sebuah projek yakni bookshelf dan website sederhana menggunakan html dan css. berharap kedepanya saya bisa membuat sebuah projek lebih.
